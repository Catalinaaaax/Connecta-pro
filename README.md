# Connecta Pro — Soluciones Integrales para tu Empresa 🚀

Sitio web corporativo moderno y totalmente responsivo diseñado para **Connecta Pro**, una agencia que ofrece servicios de Desarrollo de Software, Diseño & Marketing y Consultoría Fiscal.

## 📋 Características

- **Diseño Responsivo:** Adaptable a móviles, tablets y escritorio.
- **Interfaz Moderna:** Uso de gradientes, tarjetas con efecto "glassmorphism" y animaciones suaves.
- **Integración con WhatsApp:**
  - Botón flotante en todas las páginas.
  - Formularios de contacto y cotización que redirigen a la API de WhatsApp con mensajes pre-llenados.
- **Secciones Clave:**
  - Hero con métricas.
  - Servicios y Consultoría (con precios).
  - Paquetes comerciales.
  - Testimonios y Preguntas Frecuentes (Acordeón).
- **Sin Dependencias:** Construido con tecnologías web estándar sin frameworks pesados.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica.
- **CSS3:** Variables CSS (Custom Properties), Flexbox, CSS Grid y animaciones.
- **JavaScript (Vanilla):** Lógica para el menú móvil, modales, acordeones y validación de formularios.
- **FontAwesome / SVG:** Iconografía optimizada.

## 📂 Estructura del Proyecto

```text
/
├── index.html      # Estructura principal
├── style.css       # Estilos y diseño visual
├── script.js       # Interactividad y lógica
└── assets/         # Imágenes e iconos
```

## 🚀 Cómo ver el proyecto

1. Clona este repositorio o descarga el ZIP.
2. Abre el archivo `index.html` en tu navegador web favorito.
3. ¡Listo! No requiere instalación de servidores ni compilación.
